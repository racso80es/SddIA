# Eventos de Dominio (Domain Events)

Los Eventos de Dominio son mensajes inmutables diseñados para representar sucesos significativos que ya han tenido lugar en la aplicación, como la creación de un nuevo usuario [31], [32]. Este patrón es la piedra angular en arquitecturas dirigidas por eventos (EDA) ya que permite desacoplar los casos de uso principales de las lógicas secundarias derivadas, haciendo posible respetar estrictamente los principios SOLID [33], [34].
